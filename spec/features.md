# 已完成功能清單

## 4.1 認證模組

| 功能 | 行為說明 |
|------|----------|
| 會員登入 | Email + 密碼，取得 JWT Access/Refresh Token 存入 localStorage |
| 會員註冊 | 姓名、Email、密碼（≥8字元）、電話（選填） |
| 自動登入 | App 啟動時有 token → 呼叫 `/users/me` 還原狀態 |
| Token 自動更新 | 401 時自動用 Refresh Token 換新，最多重試一次 |
| 登出 | 清除 localStorage token + context 狀態 |
| 登入守衛 | `requireAuth()` — 未登入時阻擋並開啟登入 Modal |

## 4.2 活動探索模組

| 功能 | 行為說明 |
|------|----------|
| 首頁搜尋 | 關鍵字搜尋標題、地點、標籤 |
| 地點篩選 | 全台灣或指定縣市（22縣市＋離島） |
| 運動分類篩選 | 10 大主類別 × 多子類別 |
| 探索標籤 | 首頁快速篩選卡片（7 預設 + 自訂） |
| 探索標籤管理 | 新增、編輯、刪除、啟用/停用、拖曳排序，同步後端 |
| 活動列表 | Load More，每次 +6 筆 |
| 進階篩選 | 縣市、日期、費用區間、難度、快額滿，即時更新 |
| 搜尋高亮 | 符合關鍵字以橘色高亮 |
| 地圖視圖 | Leaflet 地圖，橘色標記，點擊 Popup 查看詳情 |
| 骨架屏 | 載入期間顯示 skeleton（700ms 模擬延遲） |

## 4.3 活動詳情模組

| 功能 | 行為說明 |
|------|----------|
| 活動詳情頁 | 大圖、狀態徽章、社團卡、日期時間地點費用、描述、標籤 |
| 報名狀態 | LIMITED：進度條（N/M）；OPEN：參加人數頭像 |
| 小地圖 | 底部嵌入 Leaflet（有座標時） |
| 報名入口 | 「立即報名」開啟 RegistrationModal |

## 4.4 報名模組

| 功能 | 行為說明 |
|------|----------|
| 三步驟報名 | Step1 組別+交通；Step2 個人資料；Step3 確認送出 |
| 組別選擇 | OPEN 模式選分組 |
| 個人資料 | 真實姓名、電話、緊急聯絡人姓名與電話 |
| 候補說明 | 額滿時顯示候補說明，仍可送出 |
| 報名成功 | Toast 通知，加入「我的活動」 |
| 取消報名 | 個人頁面取消，從 `myActivityIds` 移除 |

## 4.5 社團模組

| 功能 | 行為說明 |
|------|----------|
| 社團列表 | 社團卡片（Logo、名稱、評分、人數、標籤），骨架屏 |
| 社團頁面 | 三頁籤：社群（貼文）、活動、相簿 |
| 加入社團 | 更新 `joinedClubIds`，Toast 通知 |
| 退出社團 | 更新 `joinedClubIds`，Toast 通知 |
| 活動月曆 | 週/月/年視圖切換日曆 |
| 社團管理 | 編輯名稱/描述/Logo/標籤，管理成員（踢除），刪除社團 |
| 成員等級徽章 | 依參加活動次數顯示等級（見 [data-models.md#clubmemberrank](data-models.md#clubmemberrank)） |

**等級徽章顯示位置**
1. 社團成員列表（管理員 → 管理成員）：名字右側
2. 貼文作者名稱旁：小型 icon
3. 個人資料頁已加入社團卡片（待實作）

## 4.6 社群互動模組

| 功能 | 行為說明 |
|------|----------|
| 發文 | 選擇類型（公告/閒聊/相簿），輸入內容、上傳圖片 |
| 貼文篩選 | 依類型篩選 |
| 貼文排序 | 最新/最早切換 |
| 按讚 | 按讚/取消，即時更新（local state） |
| 巢狀留言 | 頂層 + 回覆（parentId），最多兩層 |
| 編輯/刪除貼文 | 管理員可對任意貼文操作 |
| 刪除留言 | 管理員可刪除 |

## 4.7 個人中心模組

| 功能 | 行為說明 |
|------|----------|
| 個人頁面 | 頭像、名稱、報名活動、加入社團 |
| 取消報名 | 活動卡片上取消，立即移除 |
| 退出社團 | 社團卡片上退出 |
| 編輯資料 | 修改名稱、頭像（PATCH /users/me） |
| 設定 | SettingsModal：Dark Mode、語言（佔位）、登出 |

## 4.8 通知模組

| 功能 | 行為說明 |
|------|----------|
| 通知列表 | 未讀以深色背景標記 |
| 一鍵全讀 | 全部標為已讀 |
| 點擊跳轉 | ACTIVITY → 活動詳情；INVITE/INTERACTION → 社團頁 |
| 未讀徽章 | Nav 鈴鐺顯示未讀數紅點 |

## 4.9 系統功能

| 功能 | 行為說明 |
|------|----------|
| Dark Mode | 全站 Tailwind `dark:` class，localStorage 持久化 |
| Toast | 底部彈出（success/error/info），3 秒消失 |
| 響應式 | 手機底部導覽列 / 桌面浮動 Dock |
| FAB | 浮動建立按鈕 → 選擇活動/社團/貼文 |
| 刪除確認 | 所有破壞性操作用確認 Modal（禁用 `confirm()`） |

## 4.10 訊息系統（v1）

| 功能 | 行為說明 |
|------|----------|
| 對話列表 | 對方頭像、名稱、最後訊息預覽、時間 |
| 發送訊息 | v1 純文字，圖片列未來版本 |

| 已讀回執 | 未來版本 |
